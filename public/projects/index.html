01.

Full Stack Image Recognition App

This is the final project for the Udemy Course: 'Zero to Mastery' by Andrei Neagoie. The challenge was to create an app that detects faces upon the submission of an image link. The project uses Clarifai Machine Learning Image Recognition API to detect faces. There is login & register functionality. Each account that signs up has a score that increments for each time the user submits a link that successfully detects a face. That score is stored in a database on Heroku server so that when the user logs back in, your account score is still there.

IssInfo

IssInfo is a single page application that uses the Open-Notify-API to fetch and display information about the International Space Station such as live location. This app also uses Leaflet, an open-source JavaScript library for interactive maps, to display the location of the ISS.
